# Truck Driver Guide

## Main Features
- View and accept tasks
- See assigned routes and run them live
- Post GPS for live tracking (battery-aware)
- View schedules after accepting tasks
- Vehicle info and settings
- Notifications

## Daily Flow
1. Check tasks and accept assignments
2. View today's route and start run
3. Keep app open to enable GPS updates
4. Complete stops; report incidents if any
5. Update status (On Duty/Off Duty/On Leave)

## Tips
- Grant location permission and keep battery optimization off for reliable tracking.
- Use Route Run to focus on active route only.
