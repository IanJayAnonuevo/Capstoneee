# Admin Guide

## Main Features
- Dashboard and summaries
- Manage Users (add/update staff/residents, statuses)
- Manage Routes and Schedules
- Task Management (assign/update/delete)
- Special Pick-up Requests
- Barangay Activity
- Issues and Feedback viewing

## Daily Tasks
1. Review dashboard metrics
2. Process special pick-up requests
3. Assign tasks to drivers/collectors
4. Adjust schedules and routes as needed
5. Monitor issues and feedback

## Tips
- Use bulk actions in Manage Users to speed up updates.
- Keep roles consistent (see feature analysis) and plan migration to `employees` model.

## Known Gaps / Enhancements
- Real route/team assignments replacing simulated data
- Audit log for admin actions
- Server-side filtering/pagination for large lists

See also: `FEATURES_GAP_ANALYSIS.md`.
