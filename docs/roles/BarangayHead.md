# Barangay Head Guide

## Main Features
- Submit issue reports
- Request special pick-up
- View collection schedule
- IEC materials
- Notifications and settings

## Common Tasks
1. Submit/track special pick-up requests
2. Review resident reports from your barangay
3. Coordinate schedule changes and IEC campaigns

## Tips
- Use clear descriptions and photos for issue reports.
- Share IEC tips via barangay social channels.
