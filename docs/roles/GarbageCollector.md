# Garbage Collector Guide

## Main Features
- View and accept tasks
- See routes and schedules (visible after acceptance)
- Notifications and settings

## Daily Flow
1. Review assignments and accept
2. Check schedule and prepare tools/PPE
3. Coordinate with driver/team as needed
4. Complete tasks and update status

## Tips
- Confirm task acceptance early to unlock schedules view.
- Keep notifications on for assignment changes.

## Upcoming Enhancements
- Bin-level logging (fill level, contamination)
- Team coordination view
- QR/NFC scan at collection points
